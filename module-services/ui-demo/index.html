About This Demo

This demo showcases a modular unemployment insurance system that demonstrates how different services with different technology stacks can seamlessly communicate through an API Gateway.

Protocol Conversion

Seamlessly convert between GraphQL and REST APIs through the API Gateway.

Real-time Updates

Receive real-time updates on claim status across different services.

Different Technology Stacks

Node.js/GraphQL and Python/FastAPI working together through protocol translation.

System Architecture

The system consists of three primary components communicating through well-defined interfaces:

Claimant Services

Node.js Express GraphQL React MongoDB

User-facing service for submitting unemployment claims

API Gateway

Traefik Express Middleware Docker

Integration layer that handles routing and protocol conversion

Claims Processing

Python FastAPI REST Vue.js PostgreSQL

Backend service for processing submitted claims

Demo Flow

Follow these steps to see the system in action:

1

Submit a Claim

Go to the Claimant Portal and submit a new unemployment claim.

2

View API Gateway Operations

Check the API Gateway Dashboard to see how the claim is converted from GraphQL to REST.

3

Process the Claim

Go to the Claims Processor to see the received claim and update its status.

4

Check Status Updates

Return to the Claimant Portal to see the updated claim status reflecting changes from the processor.
